  • Manually copy pasting code samples is very evil, as it goes out of date
  • It would be cool if the code were colored to be more readble - see for example the Akka Tutorials.

Details of solution
I have a GO pipeline that
  1. pulls sourcecoude from github
  2. Triggers a mvn test that
    2. scans any given java file for code snippets surrounded by the two comments //YaaSBiteSnippetStart someName and //YaaSBiteSnippetEnd
    3. extracts that code and prettifies it to html using java2html
    4. writes the code to respective files in webapp
    5. deploys this webapp to the cloud so that the prettified files are available via urls
    6. the target pages reference these files using jquery such as $('#snippetGreetingEndpoint').load('https://yaasbitesadmin.cfapps.us10.hana.ondemand.com/demotests/greetingEndpoint.html'); ,
    7. When this page is loaded, it loads the latest prettifed code and includes it above.
